TDS Virtual Teaching Assistant API

This is a simple REST API that answers questions related to the Tools in Data Science course from IIT Madras.

API Endpoint

POST https://your-deployed-api.vercel.app/api/

Request Format

{
  "question": "Should I use gpt-4o-mini or gpt-3.5-turbo?",
  "image": "base64 encoded image (optional)"
}

Response Format

{
  "answer": "Use gpt-3.5-turbo-0125...",
  "links": [
    { "url": "...", "text": "Reference link" }
  ]
}

For more information, visit the GitHub repository.
